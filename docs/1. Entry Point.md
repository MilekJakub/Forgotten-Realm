# Entry point

The `main.cpp` file is an entry point to our game. Here we can find our `int main()` function and game loop.

- Initialization
- Creating the Window
- Creating `Game` Instance
- Game Loop
- Resource Cleanup

The `Game.cpp` file contains our main class `Game`.

- Game State Management
- Game Initialization
- Input Handling
- Game Logic
- Rendering
- Cleanup
